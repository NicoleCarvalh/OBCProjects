Switch Case

Permite criar blocos condicionais como o IF, porém com um estrutura que avalia uma expressão e testa o resultado contra vários casos, executando os casos correspondentes.


É possível definir um caso padrão, que sempre é executado. Também é possível parar a execução após algum dos casos.

SWITCH expressão

CASO a : faça uma coisa
CASO b : faça outra coisa E PARE
CASO c : faça mais outra coisa POR PADRÃO: faça algo sempre

Sintaxe:
switch (expressao) {
case "a": // ...
case "b": // ...
break
case "c": // ...
default: // ... }

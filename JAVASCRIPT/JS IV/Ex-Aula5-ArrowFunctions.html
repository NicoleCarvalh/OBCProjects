- Uma forma diferente de escrever funções anônimas

- Não utiliza a palavra Function, utilizamos "=>" após os parâmetros

-Não possui seus próprios this, arguments, super e new.target

- Não possui nome próprio, precisam ser instanciadas com a utilização de uma variável pra ser atribuida ou como parâmetro de uma outra função

- Também permite retornar diretamente uma expressão, caso seka uma função de uma única linha


Exemplo

const sum = (a, b) => {

return a+b

}

const multiply = (a, b) => a * b

- Quando se tem apenas um parâmetro, não é necessário utilizar parênteses

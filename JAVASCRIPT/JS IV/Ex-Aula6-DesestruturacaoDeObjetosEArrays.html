Desestruturação é uma funcionalidade moderna que permite que a gente crie variaveis a partir de priopriedadedes de um objeto ou de elementos de um array

Com a desestruturação de objetos, podemos pegar uma parte da estrutura desse objeto e criar uma variável em cima dessa parte

No caso das priopriedadedes do objeto, as variáveis serão criadas com o mesmo nome da propriedade

No caso de elementos do array, podemos atribuir qualquer nome para a variável. Utilizamos a ordem dos itens, ou seja, os índices, para trabalhar com ele

Também pode ser utilizada de outras formas, como nos parâmetros de uma função

Exemplo:

const {name, email} = person

const [x, y] = coordinates

function hello({name}){

console.log(`Hello, ${name}`)

}

Spread Operator " ... " é um operador que permite espalhar (do inglês spread), os elementos de um objetp iterável, como um array ou uma string

Ele separa os elementos

Para que é utilizado?

  1. Passar vários parâmetros de uma única vez
  2. Clonar objetos iteráveis, ou seja, duplicar eles
  3. Fazer manipulações sem afetar o objeto original
  4. Transformar strings em arrays

Utiliza três pontos: ...iterableObject

Pode ser usado para diferentes fins. Exemplo:

let str = "Olá, Mundo"

let arr = [0, 1, 8, 2, 7]

console.log(...str) // separa as letras: O l á , M u n d o

console.log(...arr) // separa os elementos 0 1 8 2 7

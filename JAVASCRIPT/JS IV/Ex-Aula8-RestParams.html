É uma técnica que permite uma função receber qualquer número de parâmetros

Permite pegar todos os parametros que foram passados em uma função e agrupar o conjunto deles em um arrar

Podemos trabalhar dentro da função os parâmetros no formato de um array

Ou seja, permitindo manipular esses parâmetros como um array na definição da função, ele se diferencia do objeto arguments (outro nome para parâmetros)

Podemos tratar os parâmetros com os métodos dos arrays dentro da função

Caso tenhamos mais de um parâmetro, devemos colocar o rest params sempre no final na ordem dos parâmetros, sempre deverá ser o último na lista de parâmetros. Se colocarmos " function sum(...numbers, param) " teremos um erro. O certo será " funtion sum(param, ...numbers) "

Exemplo:

function sum(...numbers) {

return numbers.reduce((accum,num) = > accum + num, 0)

}
